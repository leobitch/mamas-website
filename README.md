# Christina Bradtke - Persönliche Website

## Über diese Website

Eine persönliche Ausstellungsseite, die Christina's Welt zeigt:
- **Medical Medium Ernährung** nach Anthony William
- **Ihre Liebe zu Katzen** und das Leben mit Samtpfoten
- **Persönliche Heilungsreise** von 2014 bis heute

- ❌ Keine Premium-Angebote

## Technologie

- **HTML5** mit integriertem CSS
- **Responsive Design** für alle Geräte
- **Unsplash Bilder** für hochwertige Fotografie
- **Font Awesome Icons**
- **Google Fonts**: Playfair Display, Lora, Inter

## Design-Philosophie

- **Warme, erdige Farben**: Grün, Creme, Gold
- **Große, schöne Bilder**: Fokus auf visuelle Ästhetik
- **Klare Typografie**: Elegant und lesbar
- **Keine Ablenkungen**: Fokus auf Inhalte, nicht auf Verkauf
